# ComicZone
1. buka XAMPP dam aktifkan Apache dan MySQL.
2. buka http://localhost/phpmyadmin di browser dan import file comiczonedatabase.sql yang ada di folder ke dalam phpMyAdmin.
3. masukkan file mysql-connector-j-9.7.0 yang ada di folder ke dalam Referenced Library VSCode (JAVA PROJECT/Referenced Library).
4. program siap dijalankan.
